# 法言法语黑话转换 Skill 素材整理（新标准版）

> 来源素材：《大理院判决例全书——语言风格素材》  
> 整理目标：从素材中提取“高级、少见、文言化、判牍腔、法言法语感”的表达机制。  
> 整理原则：剔除基础法学术语，如“善意第三人、法律行为、相对人、物权、效力”等，只保留真正具有风格迁移价值的表达。

---

## 一、总体风格定位

这批素材最有价值的不是现代部门法术语，而是以下几类表达：

1. **民国判牍腔**
   - 如：自应、不得遽谓、断难予以、未便、殊为误解、于法既无不合、自无……之理。

2. **文言条件判断腔**
   - 如：苟……则……、非……始得……、纵令……亦……、固……惟……、若……而……。

3. **旧式司法公文腔**
   - 如：声请、呈由、迳呈、该管审判衙门、铺保、户邻切结、准驳、照章预纳、查核其是否真确。

4. **古典否定与裁判口吻**
   - 如：无……之理、不容……、不能不……、不得不……、不得径认、不能遽视为。

5. **高密度书面动词**
   - 如：援用、准据、查照、审究、斟酌、酌定、认许、剥夺、干预、拒斥、承还、分偿、析产、回赎、顶受、找贴、夺佃、撤佃。

6. **少见高雅贬义表达**
   - 如：弁髦、借端指勒、希图短价、妄行缠讼、空言主张、空言攻击、委卸其责、越俎专擅。

## 八、可复用高级表达模板

### 模板 1：兹姑不论式搁置

```text
【争点甲】是否成立，兹姑不论；即就【争点乙】而观，亦难谓其当然发生【效果】。
```

示例：

```text
其是否确有加班事实，兹姑不论；即就微信群催促之内容而观，亦难谓其当然发生额外报酬请求之效果。
```

---

### 模板 2：断难予以法之效力

```text
【行为/习惯/主张】纵令属实，然其有碍【公共秩序/交易安全/权利安定】，断难予以法之效力。
```

示例：

```text
插队抢座之习惯纵令属实，然其有碍公共秩序与排队利益之安定，断难予以法之效力。
```

---

### 模板 3：不得遽谓

```text
仅凭【表面事实】，尚不得遽谓【法律结论】。
```

示例：

```text
仅凭对方发送“好的”二字，尚不得遽谓双方已就晚饭费用负担达成确定合意。
```

---

### 模板 4：不能仅以空言主张

```text
【主体】主张【权利/事实】，苟无【证凭/成规/明示表示】足资证明，自不能仅以空言主张。
```

示例：

```text
室友主张其享有冰箱上层独占使用之权，苟无宿舍成规或明示约定足资证明，自不能仅以空言主张。
```

---

### 模板 5：藉词—图谋私利

```text
若仅藉词【公共理由/正当名义】，实为【个人目的】之计，则自为法所不许。
```

示例：

```text
若仅藉词维护群聊秩序，实为排挤异见成员之计，则自为法所不许。
```

---

### 模板 6：无……之理

```text
【某主体】既未具备【条件】，自无【享有/请求/处分】之理。
```

示例：

```text
该同学既未参与点餐，亦未预先垫付费用，自无要求优先分配炸鸡之理。
```

---

### 模板 7：固……惟……

```text
【主张】固非无据，惟仍须以【条件】为限，不能因此径认【结论】。
```

示例：

```text
其主张“先到先得”固非无据，惟仍须以未违反公共排队秩序为限，不能因此径认其得永久占座。
```

---

### 模板 8：始得认其成立

```text
必有【事实要素】及【意思要素】并存，始得认【关系/权利/责任】成立。
```

示例：

```text
必有明确分摊费用之约定及实际参与消费之事实并存，始得认共同付款义务成立。
```

### 模板 9：不容事后再生异议

```text
【主体】既已【明示/默示表示】，即不容事后再生异议。
```

```text

若谓【错误理解】，则殊为误解。
```

## 十一、五个按新标准改写的示例

### 示例 1

大白话：

```text
他就是找借口不想还钱。
```

法言法语黑话：

```text
其所谓不能清偿之陈述，若无确切证凭足资证明，自不能仅以空言承允或空言抗辩而免其清偿之责。苟其藉词资力不足，实为迁延履行之计，则殊难认为正当。
```

---

### 示例 2

大白话：

```text
他乱插手我们宿舍的事。
```

法言法语黑话：

```text
其既非共同生活关系中之利害关系人，亦未受成员明示或默示之委任，自不得率行干预宿舍内部事务。若仅藉词维护公共秩序而妄行过问，则于规则自治之本旨殊有未合。
```

---

### 示例 3

大白话：

```text
你不能光说我答应了，得有证据。
```

法言法语黑话：

```text
是否已有承诺之意思表示，不能仅凭一造空言主张。苟无书面、言词、行为或其他确切证凭足资认定，自不得遽谓双方已有合意成立。
```

### 示例 4

## 十二、Skill 生成公式

这批素材的真正用法可以概括为：

```text
不要把普通话直接换成现代法学术语，而要把它改成“文言副词 + 裁判判断 + 程序/证据门槛 + 否定限制 + 公益/秩序抬升”的判牍腔。
```

核心生成公式：

```text
普通事实
→ “兹姑不论 / 苟 / 纵令 / 固”开头
→ 加“证凭、准据、审究、斟酌、查核”
→ 加“不得遽谓、未便径认、断难予以法之效力”
→ 最后用“自无……之理 / 自应……以为断 / 殊为误解”收束。
```

示例：

大白话：

```text
他没证据就乱说。
```

黑话化：

```text
其所称事实是否存在，兹姑不论；苟无确切证凭足资认定，自不得仅以空言主张而遽谓相对人应受其不利判断。若仍据此妄行攻击，则殊为误解。
```
