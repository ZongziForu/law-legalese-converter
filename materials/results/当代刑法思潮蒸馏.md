## 1.1 核心气质

> **现代刑法教义学骨架 + 德语理论译介腔 + 哲学方法论抬升 + 分类与反例驱动的长句论证。**

真正制造“法学黑话感”的，是以下组合：

1. 将自然人改写成规范角色；
2. 将动词改写成抽象名词；
3. 将单一因果改写成多阶段归责链；
4. 将事实争议提升为概念定位或体系协调；
5. 先否定日常理解，再提出更抽象的重构；
6. 通过条件、例外和反例削弱直接结论；
7. 使用德语概念、学派谱系和法哲学二分增加理论纵深；
8. 最终不以“对/错”结尾，而以“能否纳入某一评价结构”结尾。

# 2. 从大白话到法言法语的总生成流程

1. **抽取命题核**：谁，在什么情境下，做了什么，影响了谁，产生什么后果。
2. **主体角色化**：把“我、你、他、老板、平台”替换为“权利主体、义务主体、行为人、组织领域支配者”等。
3. **行为名词化**：把“骗、打、拿、拖、不给”转换为“欺罔性介入、侵害性作用、占有转移、履行迟延、给付义务不履行”。
4. **对象法益化**：把“钱、身体、面子、时间、秩序”提升为“财产利益、身体法益、人格评价利益、时间性协力利益、规范秩序”。
5. **事实—规范分层**：先承认生活事实，再说明事实本身不能自动产生法律评价。
6. **因果链条化**：行为 → 中间状态 → 意思/风险/支配变化 → 结果 → 归责。
7. **体系定位化**：询问它属于构成要件、违法性、罪责、义务、风险、竞合或价值层面的哪一问题。
8. **否定—重构化**：使用“并非……而是……”“关键不在于……而在于……”。
9. **条件和例外化**：加入“只有当……方可……；反之……则……”。
10. **庄严收束化**：以“因此，该问题最终表现为……”或“其规范意义在于……”收尾。

### 2.1 最小转换公式

> 【日常事实】  
> → 【规范角色】实施【名词化行为】  
> → 介入【关系/过程/状态】  
> → 导致【法益/利益/规范期待】发生变动  
> → 只有在【限定条件】下，方可纳入【评价框架】  
> → 其关键并非【表层事实】，而在于【体系性问题】。

# 3. 语言复杂化机制

## 3.1 主体抽象化：把“人”改造成规范角色

文本中的人很少只作为生活中的“某个人”出现，而会被重新界定为行为人、权利主体、义务主体、归责主体、规范对象、法益承载者、风险制造者、行为支配者、保证人、组织领域的管辖者等。

| 普通主体 | 法学化替换 |
|---|---|
| 我 | 权利主体、利益归属主体、相对人、法益承载者 |
| 你/他 | 行为人、义务违反主体、风险制造者、归责对象 |
| 老板 | 给付义务负担者、组织领域支配主体 |
| 公司 | 制度性组织体、组织风险的管辖主体 |
| 平台 | 信息配置结构、交易秩序组织者、风险分配主体 |
| 家长 | 保护义务主体、监督义务承担者 |
| 路人 | 一般社会成员、非特定义务主体 |
| 被害人 | 法益承载者、利益减损主体、规范保护对象 |

**模板：**

> 【某人】在此并非仅以经验意义上的自然人出现，而是作为【义务】的承载者、【风险】的控制者与【归责】的对象进入评价结构。

## 3.2 行为名词化：把动作改写成过程、状态和关系

| 大白话 | 名词化表达 |
|---|---|
| 骗 | 欺罔性信息输入、错误认识之诱发 |
| 打 | 侵害性作用、暴力支配行为 |
| 拿走 | 占有支配关系之转移 |
| 不还钱 | 给付义务之不履行 |
| 迟到 | 时间性协力义务之迟延履行 |
| 不救 | 防止结果发生义务之不履行 |
| 帮他做 | 对行为实现过程的促进 |
| 管不好 | 组织领域内风险控制义务之欠缺 |
| 说谎 | 信息真实性义务之违反 |
| 插队 | 对公共排序秩序的单方改写 |
| 拉黑 | 交往可能性的单方中断 |
| 放鸽子 | 约定协力期待之落空 |

**模板：**

> 【普通动作】并非仅是物理意义上的举动，而是对【利益状态/支配关系/规范期待】所实施的【侵害性/扰乱性/变更性】作用。

## 3.3 客体法益化：把“东西”和“不爽”转换为利益结构

常见转换包括：财产法益受到减损、身体完整性利益受到侵害、人格评价状态发生下降、规范所保护的利益处于危险、法益归属秩序被扰动、保护范围尚未被触及。

**模板：**

> 【对象】在此并非纯粹的物理客体，而是【某种法益】的承载媒介；其被改变的意义，应从【保护目的】而非单纯外观加以把握。

## 3.4 事实—规范二层化：事实不会自动说话

典型转换为：

> 自然事实 → 生活事实 → 法律概念化 → 规范评价。

**模板：**

> 【事实】固然已经发生，但事实发生并不当然等同于【法律判断】成立；二者之间仍须经过【概念识别】与【规范归属】的转换过程。

## 3.5 概念定位：把定义问题改造成体系位置问题

一个概念是否正确，不只看定义，还要看它位于哪个阶层、与相邻概念如何区分、承担什么功能、产生何种后果、是否造成重复或漏洞。

**模板：**

> 【概念】的意义并不取决于其字面外观，而取决于其在【体系】中的定位、与【相邻概念】的边界以及由此产生的评价后果。

## 3.6 阶层化：把一个问题拆成多层检验

通用检验次序：

1. 行为是否存在；
2. 是否符合某种类型；
3. 是否具备违法性；
4. 是否可以责备；
5. 是否有制裁必要；
6. 是否存在评价重复。

**模板：**

> 对【问题】的评价不能一次完成，而应依次检验【事实层】、【关系层】、【义务层】、【归责层】与【后果层】。

## 3.7 因果链条复杂化

常见链条：

- 欺骗：信息输入 → 认识偏移 → 意思形成受影响 → 处分行为 → 利益减损 → 归责。
- 风险：风险创设 → 风险升高 → 进入保护范围 → 风险现实化 → 结果发生 → 客观归责。
- 不作为：义务来源 → 支配可能性 → 未采取措施 → 结果发生 → 等价性判断 → 归责。
- 竞合：同一事实 → 数个类型 → 保护利益比较 → 评价内涵比较 → 是否重复 → 适用规则。

**模板：**

> 【行为】首先造成【中间状态 A】，继而改变【关系 B】，并在【条件 C】介入后使【结果 D】现实化；只有当该结果正是前述【风险/义务违反】的展开时，方能建立归责关联。

## 3.8 风险化：把“危险”变成可评价的规范对象

需要区分风险是否被容许、是否由行为人制造、是否属于通常风险、是否进入保护范围、是否在结果中现实化、第三人介入是否切断归责。

**模板：**

> 问题不在于【行为】是否伴随抽象危险，而在于其是否创设了超出社会通常容许范围的风险，以及该风险是否在具体结果中获得现实化。

## 3.9 支配化：把“能不能控制”提升为归责能力

支配可写成现实支配、支配可能性、组织支配、意思支配、风险支配、支配义务、对流程的决定性影响。

**模板：**

> 【主体】是否应承担中心责任，不取决于其是否亲自完成全部动作，而取决于其是否对【流程/风险/信息/资源】具有决定性的支配可能性。

## 3.10 义务化：把“应该做”追溯到规范来源

需要追问义务来自法律、契约、身份、前行为还是组织管辖，是保护义务还是监督危险义务，是一般义务还是特定义务，是否有履行可能性。

**模板：**

> 单纯未实施【动作】尚不足以形成归责，仍须说明行为人为何处于负有【特定义务】的地位，以及该义务是否具有防止结果发生的功能。

## 3.11 否定—重构：先拆掉常识，再建造理论

### 核心结构

- 并非……而是……
- 不是……而是……
- 不在于……而在于……
- 不能简单理解为……
- 表面上……实质上……
- 与其说……毋宁说……
- 不是没有……但……
- 固然……然而……
- 纵然……亦不能……

### 功能

1. 宣布日常语言过于粗糙；
2. 将问题转入更高层次；
3. 制造“作者掌握深层结构”的感觉；
4. 为长篇解释创造必要性。

**模板：**

> 该问题并非单纯的【事实问题】，而是【规范目的】、【概念边界】与【评价后果】之间的结构性协调问题。

## 3.12 条件化：把肯定句改写成成立条件

直接判断：“这是侵害。”

复杂判断：

> 只有当行为超出通常容忍限度，并使受保护利益发生规范性减损时，方可进入侵害评价。

常见条件工具：如果、只有、只要、除非、即使、纵然、在……前提下、以……为限、原则上、例外地、尚须、仍应、不得径行。

**模板：**

> 只有在【条件 A】、【条件 B】与【条件 C】同时具备时，【事实】才可能被评价为【概念】；反之，若仅存在【较弱事实】，则尚难径行得出相同结论。

## 3.13 程度化和阈值化：避免非黑即白

语料中对认识、风险、可能性和责任经常作程度区分，例如可能性、高度概然性、可避免性、支配程度、风险升高、侵害强度、认知能力和评价范围。

**模板：**

> 【状态】并非有无二分，而呈现从【低程度】至【高程度】的连续光谱；真正需要确定的是何种程度足以跨越规范评价的门槛。

## 3.14 反面定义：通过排除相邻概念说明自身

不直接说 X 是什么，而是通过以下方式界定：

- 如果存在 A，则不可能存在 X；
- 若不具备 B，则尚不能排除 X；
- X 不是 Y，也不是 Z，而是剩余类型；
- 通过阻却条件界定成立范围。

**模板：**

> 【概念 X】的范围可以通过反面排除加以确定：凡已具备【排除条件】者，不再属于 X；其余尚未被相邻类型吸收的情形，方进入 X 的评价范围。

## 3.15 类型学：将连续现象切割为若干“类型”

文本频繁使用一元/二元、主观/客观、支配犯/义务犯、保护法益/监督危险、特别关系/补充关系/吸收关系、事前/事中/事后、一行为/数行为、同质/异质/交错关系等二分或多分结构。

### 类型生成方法

1. 先确定区分标准；
2. 根据标准建立两类或三类；
3. 提供典型例子；
4. 检验边界情形；
5. 处理无法归类的剩余情形；
6. 必要时重新命名或合并类型。

**模板：**

> 依【区分标准】不同，可将【现象】区分为 A 型、B 型与过渡型；三者虽在外观上相近，但其规范基础与评价后果并不相同。

## 3.16 概念二分后再调和

语料不只建立对立，还常在后段说明双方各自正确一部分：价值不能从事实直接推出，但实现价值必须观察事实；规范和存在不是简单互斥，而可能是目的与手段关系；支配与义务也可被重新配置。

**模板：**

> 两种立场看似彼此排斥，实则分别回答不同层次的问题：前者处理【目的/价值】，后者处理【手段/现实】。因此，与其择一排斥，毋宁重新界定二者的功能分工。

## 3.17 设问与反问：将陈述改造成理论危机

常见形式：

- 究竟……？
- 难道……？
- 岂不是……？
- 为何……？
- 是否……？
- 何以……？
- ……的意义何在？
- ……又如何可能？

**功能：**

1. 打断平铺叙述；
2. 暗示现有理论不能自圆其说；
3. 为新的分类或重构铺路；
4. 将作者结论包装为回应难题。

**模板：**

> 如果【既有解释】成立，那么【反直觉后果】岂不是也必须被接受？问题因此不在于是否承认【概念】，而在于如何说明其界限。

## 3.18 举例—反例—边界例循环

语料常采用以下顺序：

1. 提出抽象命题；
2. 给出最容易接受的典型例；
3. 给出相反例；
4. 再给出介于两者之间的边界例；
5. 迫使概念进行修正。

**模板：**

> 在典型情形 A 中，结论似乎没有疑问；但若将事实稍作改变为 B，原有标准即出现困难；若进一步考虑 C，则可发现真正具有决定意义的并非【旧标准】，而是【新标准】。

## 3.19 长篇生活类比：用具体事物解释抽象体系

### 语料中的主要类比资源

- **空间类比**：位置、纵向空间、横向空间、观念空间、评价空间；
- **绘画与建筑类比**：画布、构图、地桩、描摹、构筑；
- **钻石与结晶类比**：体系是知识发展的结晶；
- **饮食类比**：进食顺序、咀嚼、消化、规则与生活品质；
- **机器类比**：案例放入体系机器检验；
- **语言系统类比**：主词、动词、受词构成规则系统；
- **电脑乱码类比**：原始事实必须经转换系统才能被理解；
- **自然现象类比**：苹果落地、日夜循环、落叶、地心引力；
- **流程类比**：系统输入、处理、输出；
- **中心人物类比**：谁对结果发生拥有最终决定力。

### 生成规则

1. 选择一个日常领域；
2. 找到“材料—规则—处理—结果”的对应关系；
3. 将法律概念映射为该领域的环节；
4. 最后撤回类比，说明其只用于揭示结构。

**模板：**

> 如同【日常系统】不能仅凭【最终结果】评价其合理性，【法律/制度问题】也不能跳过【中间步骤】直接得出结论。

## 3.20 学说谱系化：把观点写成历史演进

一个概念常被写成：

> 萌芽 → 提出 → 修正 → 批评 → 继受 → 转型 → 当前定位。

常用表达包括：开风气之先、逐渐形成、首度引介、往后被称为、其后陆续、一度盛行、重新处理、以新的面貌延续、回归某种思想、终结某一阶段、重新定位。

**模板：**

> 【概念】并非一开始即具有今日形态，而是在【早期命题】的基础上，经由【修正】、【反驳】与【功能转换】逐渐形成。

## 3.21 外国术语嵌入：中文概念 + 原文 + 释义

常见形式：

> 中文译名（德语原文）  
> 学者名 + 理论名  
> 中文概念 + 拉丁语/德语短语  
> 中文解释 + “亦即”再次定义

**作用：**

1. 暗示概念来自成熟理论传统；
2. 增加术语边界感；
3. 使普通读者产生距离；
4. 为同义词之间的细微差异提供合法性。

**模板：**

> 若借用【外文概念】（【原文】）的分析框架，【普通现象】可被理解为【中文抽象解释】。

## 3.22 括号增殖：主句之外再叠加限定

括号常用于外文原词、同义解释、例子、文献、时间、条文、反例和术语限定。

**效果：**

句子的主干看似简单，但括号让概念不断分叉，产生译著和论文气质。

**模板：**

> 【概念】（此处指【限定解释】，而非【相邻概念】）在【特定条件】下……

**控制规则：**

单句不宜超过两组实质性括号，否则可读性急剧下降。

## 3.23 概念同义链：反复改述而不立即前进

常见标记：换言之、亦即、也就是、简单地说、所谓……即、这表示、这同时意味着。

**作用：**

同一命题以不同抽象层次反复表述，增加篇幅和权威感，也让概念关系显得经过精密校准。

**模板：**

> 所谓【术语 A】，亦即【释义 B】；换言之，它所处理的并非【C】，而是【D】。

## 3.24 规范目的化：用“为了什么”重新定义概念

**模板：**

> 【概念】并非先于规范而自然存在，其边界取决于【制度目的】希望保护何种利益、排除何种风险并实现何种秩序。

## 3.25 功能化：询问一个区分“有什么用”

区分概念后继续追问它是否有助于检验个案、避免遗漏、促进知识传承、保障同案同判、限制责任范围、解决利益冲突、防止评价重复或填补漏洞。

**模板：**

> 某一区分是否值得维持，不取决于术语传统本身，而取决于它能否在【判断、解释、教育、限制权力】等层面承担不可替代的功能。

## 3.26 规则—例外—例外之例外

基本结构：

1. 提出一般原则；
2. 承认例外；
3. 说明例外不能无限扩张；
4. 为例外设置附加条件；
5. 必要时再建立例外中的特殊类型。

**模板：**

> 原则上，【一般规则】；但在【特殊事实】存在时，可以例外地【不同处理】。然而，该例外仍以【限制条件】为边界，不能仅因【较弱事实】即予扩张。

## 3.27 理论—案例双向验证

理论是否具有解释力，不只看抽象定义，还要观察它能否在具体事实中提出稳定标准、涵盖更多内涵、发展下位规则，并对相近案例作出可说明的差异处理。

**模板：**

> 理论是否具有解释力，不应仅从其抽象定义判断，而应观察其能否在具体事实中提出稳定标准，并对相近案例作出可说明的差异处理。

## 3.28 评价重复和规范吸收

将“算了两次”改写为：

- 同一不法内涵是否已被评价；
- 后一事实是否被前一构成吸收；
- 数个规范是否保护同一利益；
- 是否造成双重评价；
- 是否仍有未被覆盖的独立内容。

**模板：**

> 若前一评价已经完整吸收后一事实所包含的规范内涵，再次评价即可能造成评价重复；反之，若后一事实包含独立的利益侵害，则不能仅以形式上的连续性加以吸收。

## 3.29 谨慎缓和：让结论既强又可撤回

高频缓和词：基本上、原则上、似乎、可能、未必、恐怕、可以说、不无疑问、尚待检讨、还可以进一步讨论、并非当然、不能否认、至少、就此而言。

**组合方式：**

> 强判断 + 缓和词 + 保留例外。

例如：

> 该标准原则上具有解释力，但是否足以覆盖边界情形，仍有进一步检讨空间。

## 3.30 庄严收束：从个案回到体系

常见收束层级：

1. 个案结论；
2. 概念边界；
3. 体系功能；
4. 价值选择；
5. 方法论意义。

**模板：**

> 因此，该问题最终并非【个案表象】如何处理，而是【规范体系】应如何在【价值 A】与【价值 B】之间分配风险、责任与解释负担。

# 4. 页段覆盖与可迁移资源

## 4.1 第 32—41 页：破题、空间化与方法论对立

**主要资源：**

- 把时间问题改写为观念空间的转换；
- 用“位置—视野—空间”建立理论观察框架；
- 以“究竟要看什么”发动设问；
- 使用纵向/横向、时间/地理、存在/规范等二分；
- 将作者选择写成“本文不拟触及……一来……二来……”；
- 用外国理论名称和括注制造译介腔；
- 通过“归责重点不在于心理事实，而在于规范动机”完成规范化。

**可迁移规则：**

> 把普通的时间变化、地区差异或个人经历，改写成“观察位置、评价空间与理论世代”的变化。

## 4.2 第 81—90 页：体系必要性与日常类比

**主要资源：**

- 把体系比作钻石、结晶和知识资产；
- 用进食顺序、咀嚼和消化说明步骤不能省略；
- 用机器检验案例说明体系功能；
- 将体系价值分成教学、传承、判断、公平和规则形成；
- 通过两个“不处罚”案例说明同结论背后可能有不同理由；
- 以“体系化等于精致化”完成价值提升。

**可迁移规则：**

> 不要只说某个流程“有必要”，而要说明没有该流程会造成遗漏、混乱、不同情形同一处理或相同情形不同处理。

## 4.3 第 170—179 页：概念形成、电脑乱码与事实价值关系

**主要资源：**

- 将行为概念区分为先在事实与规范概念；
- 用电脑乱码和不同文字系统说明概念转换；
- 用苹果落地、日夜循环和地心引力区分自然规则与价值判断；
- 对一元论和二元论各承认一部分正确性；
- 最终将对立重写为目的与手段的关系。

**可迁移规则：**

> 先建立两个极端立场，再用“各自回答不同层次的问题”完成理论调和。

## 4.4 第 260—269 页：程度、概率与反面界定

**主要资源：**

- 用可能性、高度概然性等连续尺度替代简单有无；
- 区分希望、相信、估算、接受等心理状态；
- 通过“表现出防止结果的意思”反面界定故意；
- 以“有效经验”排除推测和过度恐惧；
- 把日常生活风险与需要避免的风险区分开。

**可迁移规则：**

> 将主观态度拆成认识程度、意欲方向、行为表现和避免动机四个变量。

## 4.5 第 349—358 页：类型边界、时间切片与概念误用

**主要资源：**

- 通过多个假设例比较一个或多个可能类型；
- 使用“如果物体是人/如果物体是物”式分支推演；
- 以“概念误用”否定事前或事后的标签；
- 将故意限定于行为时点；
- 通过第一个行为是否有效、第二个行为是否另行起意检验因果偏离；
- 以边界案例迫使概念重新定位。

**可迁移规则：**

> 将一个事件切分为准备、开始、进行、结束、事后五个时点，分别判断同一概念是否存在。

## 4.6 第 438—447 页：风险、注意义务与裁判叙事

**主要资源：**

- 区分故意与过失的主客观构造；
- 以“不被容许风险”定义行为不法；
- 将注意义务的对象、程度和主体分别识别；
- 以法律规则的规范对象排除错误归责；
- 案例叙述采用事实—起诉—一审—终审—评论的层次；
- 通过第三人利用危险源检验信赖原则。

**可迁移规则：**

> 分别询问“谁负义务、义务保护什么、危险来自何处、第三人是否介入、结果是否在义务保护范围内”。

## 4.7 第 527—536 页：理论效用、经验标准与下位规则

**主要资源：**

- 把“理论有没有用”转化为解释力、涵盖力和演绎力；
- 将简单否定称为“空洞质疑”；
- 区分自然经验、规范经验和评价性判断；
- 要求抽象概念建立下位判断规则；
- 批评“万灵概念”不能代替具体标准；
- 在理论和裁判事实之间反复往返。

**可迁移规则：**

> 每使用一个抽象词，都要追问它能否生成至少三个可操作的下位判断问题。

## 4.8 第 616—625 页：支配、义务、身份与组织

**主要资源：**

- 将主体责任区分为身份基础、支配基础和义务基础；
- 说明形式身份不等于实际组织影响力；
- 将“不作为的支配”改写为支配可能性或支配义务；
- 通过组织管辖描述企业流程；
- 最终调和存在论与规范论：价值来自价值，实现价值须观察现实。

**可迁移规则：**

> 在组织问题中，不问“职位叫什么”，而问“谁配置资源、控制信息、决定流程并能防止结果”。

## 4.9 第 706—715 页：法理谱系与保证人类型

**主要资源：**

- 从早期学者、后续修正到新分类构造学说史；
- 将义务来源区分为习惯法、因果论、功能说、保护法益和监督危险；
- 对每一种分类追问其法理依据；
- 通过“造路、买房、养犬、装机器”等例子检验类型是否过宽；
- 用“中心人物”“被害人倚赖”“防止责任”等概念竞争解释力。

**可迁移规则：**

> 解释某种责任时，至少提供三种可能基础，然后说明它们各自能解释什么、遗漏什么。

## 4.10 第 795—804 页：概念逻辑、互斥关系与评价完整性

**主要资源：**

- 将规范关系区分为特别、补充、吸收、交错、异质；
- 追问概念逻辑是否真正能解决适用问题；
- 提出法条原则上互斥，竞合属于非常态；
- 用立法漏洞和规范重叠解释例外；
- 将行为数、法益数和评价完整性分别检验；
- 以“评价重复/评价不足”作为最终判断标准。

**可迁移规则：**

> 当两个规则同时出现时，不只问谁优先，还要问二者是否同质、是否互相包含、是否各自保护独立利益以及择一是否足以完整评价。

# 5. 词库

说明：下表优先收录能跨主题迁移、能够制造法言法语气质的词，而不是罗列书中所有专门刑法概念。“程度”中的 5 表示黑话感极强，1 表示较普通。

## 5.1 教义学和法律评价词

| 词语 | 类别 | 程度 | 功能 | 常见搭配 | 可复用句式 | 错误使用 |
|---|---|---:|---|---|---|---|
| 构成要件 | 法律术语 | 5 | 将事实纳入类型判断 | 合致性、效力范围、要素 | 该事实是否具有构成要件意义，仍须结合规范目的判断。 | 把任何一般条件都叫构成要件 |
| 构成要件合致性 | 法律术语 | 5 | 表达“是否符合规则” | 客观、主观、阻却 | 外观相似并不当然意味着构成要件合致。 | 用于无类型规范的场景 |
| 违法性 | 法律术语 | 5 | 替代“是不是错的” | 阶层、阻却、实质 | 事实侵害性与违法性评价并非同一问题。 | 将道德不满直接等同违法 |
| 罪责 | 法律术语 | 5 | 替代“该不该怪他” | 原则、条件、非难 | 关键在于是否存在可非难的罪责基础。 | 与犯罪结果混同 |
| 归责 | 法律术语 | 5 | 替代“算谁的锅” | 客观、规范、结果、范围 | 因果先后并不足以建立规范归责。 | 将因果关系直接等同归责 |
| 客观归责 | 法律术语 | 5 | 用风险实现判断结果归属 | 风险创设、保护范围 | 只有不被容许的风险在结果中实现，方可肯定客观归责。 | 忽略风险是否被容许 |
| 不法 | 法律术语 | 5 | 高密度表达规范否定 | 行为不法、结果不法 | 该事实是否包含独立不法，决定能否另行评价。 | 当作一般“不好” |
| 法益 | 法律术语 | 5 | 将利益抽象为保护对象 | 侵害、保护、位阶 | 应先识别行为触及何种法益。 | 把所有不愉快都叫法益 |
| 法益位阶 | 法律术语 | 5 | 把利益排序体系化 | 冲突、衡量、保护 | 冲突的解决取决于相关法益的位阶关系。 | 将个人偏好冒充位阶 |
| 保护范围 | 法律术语 | 4 | 限制规则所覆盖的结果 | 规范、风险、目的 | 结果虽发生，但未必进入该规范的保护范围。 | 与适用范围完全混同 |
| 应罚性 | 法律术语 | 5 | 表达行为为何值得处罚 | 基础、判断、体系 | 应罚性解决的是行为是否具有制裁正当性。 | 与实际处罚必要性混同 |
| 需罚性 | 法律术语 | 5 | 表达是否有实际制裁必要 | 预防、界限、政策 | 即使具备应罚性，仍须检验需罚性。 | 不加区分地替代应罚性 |
| 阻却 | 法律术语 | 5 | 高级表达“排除成立” | 阻却违法、阻却罪责 | 特定事由可能阻却该层次的评价。 | 不说明阻却哪个层次 |
| 竞合 | 法律术语 | 5 | 处理规则或评价重叠 | 法条、想象、实质 | 问题不是事实多少，而是评价是否竞合。 | 把任何复杂情况叫竞合 |
| 评价重复 | 评价术语 | 5 | 替代“重复算账” | 避免、双重、吸收 | 再次评价可能造成规范上的评价重复。 | 将事实重复等同评价重复 |
| 评价不足 | 评价术语 | 5 | 表达单一规则未覆盖全部内容 | 完整、充分 | 仅适用前一规则可能造成评价不足。 | 无独立利益时滥用 |
| 规范吸收 | 评价术语 | 5 | 表达后一内容被前一评价包含 | 不法内涵、竞合 | 后一事实是否被前一规范吸收，取决于其是否具有独立不法。 | 仅凭时间连续认定吸收 |
| 归责界限 | 法律术语 | 5 | 把责任边界问题化 | 风险、结果、责任 | 真正需要确定的是结果归责的规范界限。 | 没有边界争议时空用 |
| 法律效果 | 法律术语 | 4 | 替代“最后会怎样” | 不同、相同、发生 | 概念定位不同，将导致不同法律效果。 | 与社会后果混同 |
| 规范评价 | 法学通用词 | 5 | 将事实判断转入法律层次 | 进入、完成、对象 | 生活事实尚须经过规范评价。 | 对纯事实描述滥用 |

| 意思形成过程 | 抽象词 | 5 | 把“做决定”过程化 | 介入、自由、错误 | 欺罔介入了相对人的意思形成过程。 | 任何建议都说成介入 |

# 6. 句式模板库

## 6.1 否定—重构模板

1. **事实转规范**  
   > 该问题并非单纯的【普通事实】问题，而应置于【规范框架】与【价值目标】之间的结构性关系中加以评价。

2. **外观转本质**  
   > 表面上，【事实】只是【日常描述】；实质上，它涉及【法益/义务/风险】如何在制度内部被重新分配。

3. **判断重心转移**  
   > 其关键不在于【表层现象】是否发生，而在于该现象能否被纳入【归责/评价】结构并承担相应规范后果。

4. **拒绝自然主义**  
   > 【概念】不能仅依其物理外观加以理解，而须根据其在法秩序中的【目的与功能】确定。

5. **不是没有但不足**  
   > 【事实】不是没有评价意义，但单凭该事实尚不足以完成【结论】，仍须结合【附加条件】判断。

6. **与其说毋宁说**  
   > 与其说这是【旧问题】，毋宁说它是【新问题】在【具体场景】中的表现。

7. **反直觉重构**  
   > 看似决定性的【因素 A】，在规范层面未必具有决定意义；真正具有区分力的是【因素 B】。

8. **概念误用批评**  
   > 严格说来，将【现象】称为【概念】是一种概念误用，因为【必要条件】并未具备。

## 6.2 条件和限定模板

9. **三条件成立**  
   > 只有当【条件 A】、【条件 B】与【条件 C】同时具备时，方可认定【结论】。

10. **反向排除**  
    > 反之，若【排除事实】存在，则即使【严重结果】已经发生，亦不能径行肯定【结论】。

11. **原则例外**  
    > 原则上，【一般规则】；但在【特殊情形】下，可以例外地【处理】。该例外仍以【限制条件】为边界。

12. **程度门槛**  
    > 【状态】并非有无二分，而存在程度差异；只有达到【阈值】时，才进入【规范评价】。

13. **必要非充分**  
    > 【事实 A】至多构成【结论】的必要条件，而非充分条件，尚须补充【条件 B】。

14. **即使也不能**  
    > 即使【事实】足以引发强烈直觉，亦不能因此省略【体系检验】。

15. **除非结构**  
    > 除非能够证明【附加联系】，否则【事实 A】与【结果 B】之间仍欠缺规范归属基础。

16. **以……为限**  
    > 【规则】的适用应以【目的/范围】为限，不能扩张至【排除领域】。

## 6.3 主体和行为抽象化模板

17. **规范角色**  
    > 【某人】在此并非仅是生活意义上的行动者，而是【义务】的承载者、【风险】的控制者与【评价】的对象。

18. **行为名词化**  
    > 【普通动作】应被理解为行为人对【利益/法益/关系】所实施的【侵害性/变更性/支配性】作用。

19. **组织角色**  
    > 【管理者】是否承担中心责任，不取决于其是否亲自实施，而取决于其对【组织流程】是否具有决定性支配。

20. **不作为**  
    > 单纯未实施【动作】尚不足以归责，仍须说明行为人为何负有防止【结果】发生的特定义务。

21. **意思介入**  
    > 【信息行为】通过改变相对人的认识状态，介入其意思形成过程，并最终影响【处分/选择】自由。

22. **占有关系**  
    > 【拿走/移动】的规范意义不在于物理位置改变，而在于占有支配关系被未经授权地重新配置。

## 6.4 因果、风险和归责模板

23. **完整链条**  
    > 【行为】首先引起【中间状态 A】，继而改变【关系 B】，最终导致【结果 C】；只有该结果是前述风险的现实化时，方可归责。

24. **因果不等于归责**  
    > 【行为】与【结果】之间纵有事实上的条件关系，也不当然意味着规范归责成立。

25. **风险创设**  
    > 应先判断行为是否创设了超出社会通常容许范围的风险，再判断该风险是否在具体结果中实现。

26. **第三人介入**  
    > 第三人的介入是否切断归责，取决于其行为是独立开启新风险，还是仅利用既存危险状态。

27. **保护范围**  
    > 即使结果属于行为风险的实现，仍须确认该结果是否处于相关规范意图保护的范围内。

28. **结果偶然性**  
    > 若结果仅是与行为偶然相伴的事实，而非其规范风险的展开，则不宜纳入行为人的责任范围。

29. **避免可能性**  
    > 归责的重点不只是结果能否避免，而是行为人是否具备规范上可期待的避免可能性。

30. **风险比较**  
    > 判断风险是否被容许，应综合比较【行为利益】、【危险程度】与【防止成本】，而不能只看结果严重性。

## 6.5 体系、分类和定位模板

31. **体系定位**  
    > 【概念】的意义并不取决于其名称，而取决于其在【体系】中的位置及其承担的功能。

32. **多层检验**  
    > 对【问题】应依次从【事实层】、【类型层】、【义务层】、【归责层】与【后果层】展开检验。

33. **类型区分**  
    > 依【区分标准】不同，可将【现象】分为 A 型、B 型与过渡型，三者的规范基础和后果各不相同。

34. **相邻概念**  
    > 【概念 A】与【概念 B】虽在外观上相近，但前者以【标准 A】为基础，后者则以【标准 B】为基础。

35. **剩余类型**  
    > 无法被 A 型与 B 型完整涵盖的情形，不宜勉强归类，而应检讨是否存在独立的第三类型。

36. **功能检验**  
    > 该区分是否有维持必要，取决于它能否解释【差异后果】并提供可操作的判断标准。

37. **体系漏洞**  
    > 若采【既有解释】，将导致【特定情形】既不能纳入 A，也不能纳入 B，从而产生体系漏洞。

38. **重复与不足**  
    > 适用 A 是否已经充分评价全部事实，决定继续适用 B 是造成评价重复，还是避免评价不足。

## 6.6 论辩、设问和反例模板

39. **反问压力**  
    > 如果【理论前提】成立，那么【荒谬后果】岂不是也必须被接受？

40. **意义何在**  
    > 如果【两种类型】最终产生完全相同的判断，那么维持该区分的规范意义何在？

41. **典型—边界**  
    > 在典型情形中结论似乎明确，但只要稍微改变【事实因素】，原有标准即显得不足。

42. **二重假设**  
    > 由此可以提出两种假设：若非【理论 A】的概念界定有误，即是【标准 B】不足以说明该现象。

43. **先承认后批评**  
    > 【观点】固然指出了【合理部分】，但其无法解释【边界情形】，因此尚不能作为完整理论基础。

44. **批评过苛**  
    > 若因此断言【理论】全无价值，未免过苛；较妥适的处理是限制其功能范围。

45. **空洞质疑**  
    > 仅以【概念过于抽象】为由否定其效用，若未提出替代标准，仍近于空洞质疑。

46. **反例修正**  
    > 反例并不必然推翻规则，但至少迫使规则增加【限定条件】或重新说明其适用范围。

## 6.7 哲学和外国法模板

47. **存在与规范**  
    > 从存在层面看，【事实】具有其自身结构；但从规范层面看，该结构仍须经过价值选择方能转化为法律意义。

48. **二元调和**  
    > 两种立场并非绝对互斥：前者回答价值从何而来，后者回答价值如何在现实中实现。

49. **人类图像**  
    > 该理论背后预设的是一种【自主/受制约/角色化】的人类图像，而这一预设决定其责任观。

50. **外国概念嵌入**  
    > 若借用【外文概念】（【原文】）的分析框架，【普通现象】可以被理解为【中文抽象释义】。

51. **目的决定边界**  
    > 【概念】并非先于制度而自然存在，其边界是由【规范目的】塑造的。

52. **概念转换**  
    > 原始事实必须经由【概念形成体系】转换，方能成为可被法律评价的对象。

## 6.8 谨慎判断和收束模板

53. **有限肯定**  
    > 该见解基本上具有解释力，但能否覆盖【边界情形】，仍有进一步讨论余地。

54. **尚难认定**  
    > 现有事实至多支持【弱结论】，尚难径行认定【强结论】。

55. **似乎但未必**  
    > 初步看来，【结论】似乎成立；然而，若考虑【新因素】，则未必如此。

56. **至少**  
    > 即使无法全面接受该理论，至少可以肯定其在【特定功能】上的价值。

57. **总结提升**  
    > 因此，该问题最终并非【个案】如何处理，而是体系应如何分配【风险、责任与解释负担】。

58. **规范意义**  
    > 该区分的规范意义，在于使相同情形获得相同处理，并使不同情形能够被说明为不同。

59. **功能收束**  
    > 概念精致化并非语言游戏，而是为了限制判断恣意并提高评价的可检验性。

60. **开放结论**  
    > 至于该标准是否足以成为一般规则，仍须通过更多边界案例检验。

- 输出特征：用极端庄严语言描述奶茶、迟到、抢遥控器等小事

# 8. Skill 实现建议

## 8.1 输入解析字段

```yaml
主体: 谁实施或未实施行为
相对人: 谁受到影响
行为: 做了什么或没有做什么
对象: 钱、物、身体、时间、信息、秩序等
结果: 损失、不便、危险、失望、冲突
关系: 合同、身份、组织、社交、家庭、公共秩序
主观状态: 故意、疏忽、不知道、误解
目标风格: 教义学 / 德日腔 / 古典腔 / 哲学腔 / 搞怪
膨胀倍率: 1—5
```

## 8.2 转换算法

```text
第一步：保留原命题的事实核，不得改变谁做了什么。
第二步：为主体分配规范角色。
第三步：将核心动词名词化。
第四步：把对象提升为利益、法益、秩序或规范期待。
第五步：补充一个中间过程，例如认识、意思、支配、风险或履行。
第六步：区分事实因果与规范归责。
第七步：加入否定—重构结构。
第八步：加入至少一个条件限制和一个反向情形。
第九步：按参数决定是否加入外国术语、哲学二分或长篇类比。
第十步：以体系、价值或归责问题收束。
```

## 8.3 词语注入规则

### 低强度

每句注入 1—2 个抽象词：义务、利益、关系、范围、标准、评价。

### 中强度

每句注入 2—4 个教义学词：法益、归责、风险、规范期待、构成要件、体系定位。

### 高强度

增加哲学和外国法：存在论/规范论、人类图像、价值体系、Tatherrschaft、Normativismus、功能论、概念形成体系。

## 8.4 句法扩展示例

原句：

> 他骗了我。

一阶扩展：

> 行为人向相对人提供虚假信息。

二阶扩展：

> 行为人通过虚假信息输入，使相对人的认识状态发生偏移。

三阶扩展：

> 行为人通过欺罔性信息输入介入相对人的意思形成过程，使其基于错误认识作出决定。

四阶扩展：

> 该问题并非单纯的信息不真实，而在于欺罔性输入是否通过认识错误介入意思形成过程，并导致受保护利益发生可归责的减损。

五阶扩展：

> 从规范论的评价框架观察，行为人的欺罔性信息输入并非仅是经验世界中的言语事实，而是对相对人意思形成自治的规范性介入；只有当该介入经由认识错误、处分决定与利益减损形成连续的归责链条时，方可认为其突破一般交往风险的容许边界。

## 8.5 防止“伪高级”的约束

Skill 应避免：

1. 只堆术语，不说明术语之间的关系；
2. 将因果关系、归责、责任、义务混为一谈；
3. 每句话都使用“并非……而是……”；
4. 外文术语与中文解释不对应；
5. 对任何小事都无条件认定“法益侵害”；
6. 用“体系、结构、机制”反复替换但没有新信息；
7. 条件过多导致原命题完全消失；
8. 搞怪模式伪装成真实法律意见；
9. 虚构法律原则、条文或学者观点；
10. 把文本的刑法概念机械迁移成严肃法律结论。

# 9. 十个可直接嵌入 Skill 的核心句式

1. 该问题并非单纯的【事实】问题，而是【规范期待】、【利益保护】与【风险分配】之间的结构性协调问题。

2. 【主体】在此并非仅作为生活意义上的行动者出现，而是作为【义务】的承载者、【风险】的控制者与【归责】的对象。

3. 【动作】的意义不在于其物理外观，而在于其是否改变了【支配关系/利益状态/意思形成过程】。

4. 只有当【行为】超出通常社会生活所容许的限度，并导致【利益】发生规范性减损时，方可进入【评价】。

5. 【行为】与【结果】之间纵有事实联系，也不当然意味着规范归责成立。

6. 表面上，【事件】只是【日常冲突】；实质上，它涉及【法益保护】与【行为自由】之间的界限划定。

7. 若采【既有解释】，则【荒谬或反直觉后果】岂不是也必须被接受？

8. 【概念】是否有保留必要，取决于它能否提出可操作的下位规则，并对相近情形作出可说明的差异处理。

9. 两种立场看似互斥，实则分别回答【价值从何而来】与【价值如何实现】两个不同层次的问题。

10. 因此，该问题最终并非【个案表象】如何处置，而是规范体系应如何分配【风险、责任与解释负担】。

# 10. 大白话 → 法言法语黑话示例

## 示例 1：迟到

**大白话：**

> 你迟到了。

**法教义学化：**

> 行为人未能在约定的时间框架内完成到场义务的现实化，致使相对人基于协作关系所形成的时间性履行期待发生落空。该问题并非单纯的钟点偏差，而是时间性协力义务是否被适当履行的问题。

## 示例 2：骗买东西

**大白话：**

> 他骗我买了这个东西。

**德日理论腔：**

> 行为人通过欺罔性信息输入介入相对人的认识与意思形成过程，使其在事实认知发生偏移的基础上作出财产处分。其规范意义不在于交易外观是否成立，而在于意思决定自治是否因错误诱发而受到可归责的减损。

## 示例 3：老板不发工资

**大白话：**

> 老板不给我发工资。

**体系化表达：**

> 给付义务负担者未依劳动交换关系所形成的制度性期待实现工资给付，使劳动者通过持续劳务所取得的对价请求利益处于未实现状态。该问题不能被简化为金钱迟延，而应置于劳动风险与对价利益分配的规范结构中观察。

## 示例 4：拿走东西

**大白话：**

> 他把我的东西拿走了。

**法益化表达：**

> 行为人未经利益归属主体同意，单方改变特定物的占有支配状态，使原有事实控制关系发生中断。其评价重点不在于物理位置的移动，而在于排他性支配关系是否被未经授权地重新配置。

## 示例 5：群里阴阳怪气

**大白话：**

> 群里有人阴阳怪气我。

**人格法益腔：**

> 行为人以具有隐晦贬抑指向的表达介入群体交往空间，使特定主体的社会评价利益处于被削弱的风险之中。表面上该行为仅属言辞摩擦，实质上则涉及人格利益保护与表达自由边界的重新划定。

## 示例 6：偷喝奶茶

**大白话：**

> 室友偷喝了我的奶茶。

**荒谬搞怪腔：**

> 行为人未经饮品利益归属主体授权，对特定奶茶实施了消费性占有支配，并由此使原权利主体的排他饮用可能性及其卫生信赖状态发生不可逆的规范性减损。该问题并非区区一口液体的物理消失，而是共同居住关系中饮食边界与人格尊重秩序的局部崩解。

## 示例 7：不回消息

**大白话：**

> 他一直不回我消息。

**规范期待腔：**

> 相对人持续未对既有信息输入作出回应，使发信主体基于稳定交往关系所形成的最低限度沟通期待处于悬置状态。不过，单纯沉默尚不足以当然产生回复义务，仍须考察双方关系、既往互动与具体事项的紧迫程度。

## 示例 8：插队

**大白话：**

> 那个人插队。

**公共秩序腔：**

> 行为人未经其他等待主体同意，单方重构公共资源取得的时间排序，使原有先后秩序及其所承载的平等期待受到扰动。其不当性不只在于获得便利，而在于将个人紧迫性凌驾于共同排序规则之上。

## 示例 9：项目甩锅

**大白话：**

> 项目出问题后，领导把锅甩给我。

**组织归责腔：**

> 在项目风险现实化之后，组织领域的决策支配者试图将原本由资源配置、流程控制与决策权限所生成的责任，单向转移给仅承担局部执行任务的成员。该问题的核心不在于谁最后接触了工作成果，而在于谁对风险形成过程具有决定性组织管辖。

## 示例 10：抢遥控器

**大白话：**

> 他抢走了遥控器。

**哲学化搞怪腔：**

> 行为人通过对遥控装置的排他性事实控制，取得了对共同视听内容选择过程的即时支配，使其他共同生活成员的节目决定可能性被结构性压缩。从规范共同体的角度看，这并非单一物件的位移，而是家庭娱乐治理权发生未经协商的集中化。

# 11. 质量检查清单

生成完成后，应逐项检查：

- [ ] 是否保留原始事实核，没有改变人物和事件？
- [ ] 主体是否至少完成一次角色化？
- [ ] 核心动词是否完成名词化或过程化？
- [ ] 是否明确了利益、法益、规范期待或关系状态？
- [ ] 是否区分事实发生与规范评价？
- [ ] 因果关系是否被展开为至少两个中间环节？
- [ ] 是否使用了一个否定—重构句式？
- [ ] 是否设置了条件、例外或反向情形？
- [ ] 是否说明概念在体系中的位置或功能？
- [ ] 是否避免重复堆砌“体系、结构、机制”？
- [ ] 外文术语是否真实且释义准确？
- [ ] 搞怪模式是否明确保持为修辞游戏，而非法律意见？
- [ ] 最终句是否回到风险、责任、价值或解释负担？
- [ ] 是否出现评价重复或同义反复而没有新增信息？
- [ ] 句子虽复杂，但读者能否还原原始命题？

# 12. 结论性规则

本语料最值得写入 skill 的，不是某几个刑法术语，而是以下生成规律：

1. **事实必须先被概念化，才被评价。**
2. **人应被写成角色，动作应被写成过程。**
3. **结果不直接归人，必须经过风险、义务或支配的中介。**
4. **简单二分应被拆成阶层、类型、程度和例外。**
5. **任何直觉结论都可以用“并非……而是……”重新定位。**
6. **任何肯定句都可以改造成“只有当……方才……”的条件句。**
7. **任何分类都要通过反例、边界例和功能后果检验。**
8. **概念的高级感来自其体系位置，而非词语生僻程度。**
9. **德日法学腔来自原文括注、学说谱系和概念边界敏感，而不是随机塞德语。**
10. **最典型的收束方式，是把个案提升为规范体系如何分配风险、责任和解释负担。**

### 开题

究竟、问题在于、首先需要澄清、在进入讨论之前、若欲理解、就此而言。

并非、不是、不能简单理解为、尚不足以、并不当然、未必、殊非无疑。

### 重构

而是、毋宁、真正关键在于、应置于、应理解为、其规范意义在于。

### 条件

如果、只有当、除非、以……为限、在……前提下、原则上、例外地。

### 让步

固然、即使、纵然、诚然、虽说、不可否认。

### 递进

首先、继而、进而、乃至于、最终、在此基础上、如此一来。

### 解释

换言之、亦即、也就是说、所谓、简单地说、这表示、这同时意味着。

### 对照

反之、相对地、与此不同、另一方面、前者……后者……。

因此、因而、由此可见、归根结底、总之、据此、最终表现为。
