# 蒸馏前设想阶段：法言法语高级表达、生僻词与黑话化资源整理

> 用途：整理在正式蒸馏书籍/语料之前，已由对话中推演出的“法言法语黑话转换 skill”表达资源。  
> 定位：这不是根据某一本书抽取的蒸馏报告，而是早期构思阶段形成的**通用表达资源库**，可作为后续 skill 的基础词库、参数体系和模板底稿。  
> 目标：帮助 agent 在进一步整理正式蒸馏素材时，保留早期已经形成的高级表达、生僻词、句式、参数和部门法黑话框架。

---

## 0. 总体思路

早期设想的核心不是“把普通话翻译成法律术语”，而是制作一个：

> **法律学术黑话膨胀器 / 法言法语不说人话生成器**

它的基本任务是：

```text
日常事实
→ 规范事实
→ 主体抽象化
→ 行为名词化
→ 关系链条化
→ 体系定位化
→ 价值判断化
→ 归责/效力/边界收束
```

其核心效果是：

> 明明一句话可以说完，却故意把每个普通词都升格为一个理论问题、制度问题、规范评价问题或价值冲突问题。

## 1. 大白话复杂化的基础机制

### 1.1 主体抽象化

把生活中的“我、你、他、别人、老板、员工、平台、用户”等，替换为规范结构中的功能主体。

| 普通主体 | 法言法语替换 |
|---|---|
| 我 | 权利主体、法律效果承受者、意思形成主体、规范保护主体 |
| 你 / 对方 | 相对人、义务主体、受领人、规范对象 |
| 他 | 行为人、责任主体、风险创设主体 |
| 被骗的人 | 受骗人、处分主体、意思形成主体、财产处分主体 |
| 老板 / 公司 | 用人单位、组织管理权限主体、资本组织权力主体 |
| 员工 | 劳动者、从属性地位主体、职业安定利益主体 |
| 消费者 | 交易相对人、消费利益主体、识别利益主体 |
| 平台 | 数据控制主体、交易秩序组织者、访问控制主体 |
| 作者 / 创作者 | 著作权人、表达利益主体、创作激励主体 |
| 普通人 | 法律交往参与者、法秩序中的规范主体 |

**通用句式：**

```text
【普通主体】并非仅以其生活身份出现，而应被理解为【规范身份】，其行为由此进入【法律效果/归责/信赖/效力】的判断框架。
```

---

### 1.2 行为名词化

把普通动词改造成抽象名词、过程、状态或关系变动。

| 普通动词 | 法言法语替换 |
|---|---|
| 骗 | 欺罔性信息输入、对意思形成过程的欺罔性介入 |
| 给钱 | 财产处分行为、财产性给付、财产利益转移 |
| 拿走东西 | 占有支配关系的转移、事实支配状态的变更 |
| 打人 | 对身体法益的侵害性作用、强制性身体介入 |
| 威胁 | 对意思决定自由的强制性压迫 |
| 答应 | 意思表示、承诺表示、合意形成 |
| 反悔 | 撤回、撤销、效力消灭、意思决定变更 |
| 不回消息 | 沉默、消极不作为、回应性意思表示之缺位 |
| 开除员工 | 对劳动关系持续性的单方终止 |
| 抄别人文章 | 对受保护表达的未经授权利用 |
| 发别人照片 | 对可识别人格形象的公开化利用 |
| 插队 | 对既有等待序列的非合意性嵌入 |
| 不给钱 | 给付义务之不履行、对待给付结构破坏 |
| 甩锅 | 责任归属的外部转嫁、风险承担结构的规避 |

**通用转换公式：**

```text
动词
→ 对【意思/权利/利益/法益/法律关系/占有状态/规范效果】的
【形成/变动/归属/侵扰/限制/消灭/削弱/实现】
```

---

### 1.3 因果链条化

把“A 导致 B”改造成多环节规范链条。

#### 诈骗链条

```text
欺罔行为
→ 认识错误
→ 意思形成自由受侵扰
→ 基于错误形成处分意思
→ 财产处分行为
→ 财产法益减损
→ 结果归责
```

#### 民法欺诈链条

```text
欺罔性信息输入
→ 相对人认识错误
→ 意思形成自由受侵扰
→ 基于错误作出意思表示
→ 法律行为效力瑕疵
→ 撤销权或效力控制问题
→ 信赖利益/履行利益分配
```

#### 违约链条

```text
合同成立
→ 给付义务发生
→ 清偿期届至
→ 给付义务未履行
→ 债权人给付利益不实现
→ 违约责任评价
→ 继续履行/损害赔偿/解除合同
```

#### 行政处罚链条

```text
行政机关作出处罚决定
→ 公权力介入私人权利领域
→ 需具备职权依据
→ 需满足事实认定与程序保障
→ 需接受比例原则与裁量控制
→ 否则产生合法性瑕疵
```

---

### 1.4 体系定位化

把“小事”放进大框架。

| 普通说法 | 体系化表达 |
|---|---|
| 这事不对 | 该问题应置于规范评价结构中加以判断 |
| 这不只是小事 | 该行为并非单纯生活伦理层面的失范 |
| 不能乱罚款 | 该行为涉及行政权介入私人权利领域的合法性边界 |
| 不能随便开除员工 | 该行为涉及劳动关系从属性、解雇保护与职业安定利益 |
| 不能抄别人的作品 | 该行为涉及表达保护、权利边界与公共传播利益 |
| 插队不对 | 该行为破坏了公共资源分配中的程序性期待和微观正义结构 |

**常用开头：**

```text
从【制度/体系/规范结构】的层面观察……
若置于【某一理论框架】中理解……
该问题并非事实判断问题，而是规范评价问题。
该行为不宜被孤立地理解为【普通事实】，而应被放置于【框架】之中加以评价。
```

---

### 1.5 否定—重构

这是最重要的“不说人话”结构。

```text
X 并非单纯意义上的 Y，而是在 Z 评价框架下呈现为……
问题的关键不在于是否发生了 A，而在于 A 是否能够被规范性地归入 B。
不能仅从事实层面理解该行为，而应从构成要件定型性/权利边界/制度目的的角度加以把握。
该问题表面上是 A，实质上则是 B 与 C 之间的规范张力。
```

---

### 1.6 条件化表达

把“是/不是/可以/不可以”改写为“只有当……方可……”。

```text
只有当【行为】超出【合理限度】并对【权利主体】造成规范意义上的利益减损时，方可进入侵权评价。
若【依据/程序/证据】付之阙如，则该结论难以获得规范证成。
在【构成条件】尚未具备的情形下，未可遽谓其当然成立。
```

## 2. 早期参数体系

早期已经形成的参数体系如下，后续 skill 可以继续保留。

| 参数 | 说明 |
|---|---|
| 黑话膨胀倍率 | 控制扩写长度和概念堆叠程度 |
| 法教义学密度 | 控制构成要件、归责、请求权、效力、权利边界等术语密度 |
| 名词化程度 | 控制普通动词转化为抽象名词结构的强度 |
| 句法复杂度 | 控制长句、嵌套从句、转折和限定条件 |
| 文言文化程度 | 控制“盖、是故、准此以观、殊难谓”等老派法言用量 |
| 法谚使用程度 | 控制中外法谚、原则性短句使用 |
| 外国法炫技程度 | 控制德日法、拉丁法谚、外文括注使用 |
| 哲学介入程度 | 控制康德、黑格尔、罗尔斯、哈贝马斯等哲学化表达 |
| 价值判断强度 | 控制正当性、价值基础、制度目的、利益衡量 |
| 部门法口味 | 刑法、民法、行政法、知识产权法、公司法、劳动法、诉讼法等 |
| 外观庄严程度 | 控制是否像论文、专著、判词、公牍 |
| 搞怪荒谬感 | 控制是否把鸡毛蒜皮小事过度法学化 |
| 成语使用频率 | 控制成语、四字格、老派中文论文腔 |
| 生僻词使用频率 | 控制阙如、罅隙、证成、型塑等冷僻词密度 |
| 历史考据介入程度 | 控制沈家本式“考诸旧制、刑名沿革、名实辨析”味道 |
| 法言法语强度 legalese_level | 1—5，控制表达整体“不像人话”的浓度 |
| 浮夸风险 exaggeration_risk | 1—5，控制表达是自然高级还是娱乐化炫技 |

## 5. 部门法黑话母题

### 5.1 刑法

关键词：

```text
构成要件符合性、违法性、有责性、实行行为、法益侵害、行为无价值、结果无价值、客观归责、规范保护目的、风险创设、风险实现、构成要件定型性、意思支配、认识错误、处分意识、财产法益、因果流程、归责关联、刑法评价对象
```

句式：

```text
该行为的可罚性并不取决于日常意义上的道德谴责强度，而取决于其是否在构成要件符合性、违法性与有责性的递进判断中，形成足以归责于行为人的法益侵害风险及其现实化结果。
```

### 5.2 民法

关键词：

```text
请求权基础、法律关系、给付义务、抗辩权、意思表示、信赖利益、履行利益、归责事由、私法自治、法律行为、效力评价、权利发生、权利障碍、权利消灭、权利抗辩
```

句式：

```text
该问题并非单纯的事实损害填补问题，而应首先回到请求权基础的检索结构之中，依次考察权利发生、权利障碍、权利消灭及权利抗辩等规范层次。
```

## 8. 早期自造示例

### 8.1 诈骗

普通：

```text
诈骗就是骗别人，让别人误以为是真的，然后把钱给你。
```

黑话：

```text
诈骗罪的成立并非仅以行为人实施了某种事实层面的不实陈述为充分条件，而是要求该欺罔性信息输入在规范意义上介入相对人的意思形成过程，并使其基于受瑕疵支配的认识状态作出财产处分，进而导致财产法益发生非自愿性减损。
```

### 8.2 盗窃

```text
盗窃罪并非对财物空间位置变动的单纯事实描述，而是行为人在排除权利人意思参与的状态下，破坏他人对特定财物既有的占有支配关系，并通过秘密性、非同意性的取得行为建立新的事实支配状态。
```

### 8.3 合同违约

```text
合同违约并非单纯表现为债务人未实施某一事实行为，而是其未依合同所确定的给付义务内容，在约定或法定履行期限内实现债权人给付利益的满足。
```

### 8.4 插队

```text
插队行为并非身体位置的单纯前移，而是行为人对既有等待序列的非合意性嵌入，并由此造成公共资源分配结构中的程序性罅隙。
```

### 8.5 吃饭不给钱

```text
餐饮消费后拒绝支付价款，并非单纯意义上的生活失礼或道德瑕疵，而是行为人在受领餐饮服务并取得相应消费利益之后，拒绝履行其基于服务合同关系所负担的金钱给付义务，从而破坏了交易秩序中给付与对待给付之间的规范均衡。
```

## 9. 可作为后续整理的原则

1. 早期资源不按书籍来源归类，而按功能归类。
2. 词语必须记录“功能”，否则容易乱用。
3. 对难以分类但明显高级、晦涩、文邹邹的词，先放入“待分类高深词”，不得丢弃。
4. 运行时不应把全库塞进上下文，而应按功能标签召回小词包。
5. skill 的真正核心不是模仿某个作者，而是掌握“复杂化、抽象化、法学化”的表达能力。
